Dueling Conv1D DQN Trading Agent

A deep reinforcement learning agent for algorithmic trading built in PyTorch. Uses a Dueling Conv1D Q-Network with Double DQN, N-step returns, and symmetric reward shaping to learn long-only trade entries and exits directly from OHLC price data. Trained for 150,000 steps on AAPL with CUDA GPU acceleration. Outperformed buy-and-hold by +14.30% on a held-out test set (18.15% vs 3.85%), with a 70% win rate across 20 trades — evaluated on data the agent never saw during training.

Training: 150k steps  ·  Asset: AAPL  ·  Peak profit: +870%
