Dueling Conv1D DQN Trading Agent

A deep reinforcement learning agent for algorithmic trading built in PyTorch. Uses a Dueling Conv1D Q-Network with Double DQN, N-step returns, and symmetric reward shaping to learn long-only trade entries and exits directly from OHLC price data. Trained for 150,000 steps on AAPL with CUDA GPU acceleration. Outperformed buy-and-hold by +14.30% on a held-out test set (18.15% vs 3.85%), with a 70% win rate across 20 trades.

Training: 150k steps  ·  Asset: AAPL  ·  Alpha vs B&H: +14.30%

GPU Cluster Scheduler — RL Agent

Built at a hackathon — reused the Dueling DQN architecture from the trading agent project above to meet the time constraint. A Dueling DQN agent that learns per-job GPU allocation on a shared, capacity-constrained cluster, trained under a scarcity signal derived from real Alibaba PAI cluster job traces. Balances GPU cost against deadline risk across a 9-dimensional state space and 5 discrete allocation actions, using N-step Double DQN with a replay buffer and target network. Evaluated against FCFS, Always-Max, and Priority heuristics on the same shared 32-GPU pool and job queue. Ranked Gold.

Deadline success: 87.0%  ·  vs. best baseline: +17.0 pts  ·  Cost: -$42.49  ·  Result: Gold
